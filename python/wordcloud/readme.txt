
1. Git clone
2. You will need python3 installed
3. Best to make a virtual environment
4. You will need to install the requirements
    'pip install requirements.txt'
5. To open :
    ' python imagecreate.py'

    You will be asked for a few answers:
        1. what is the filename for the text you want to use?
        2. What is the filename for the image you want to use?
        3. What background color do you want?
        4. What do you want the image saved as?

    The filenames entered can be  filename.txt if the file is int he same folder.
    If the file is in a different folder you should include the path to the file: /home/user/docs/filename.txt
